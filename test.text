\textbf{Question 1} \\ 
$\displaystyle\int^{3}_{1} \frac{1}{2 x + 2}\ dx = \left[\frac{1}{2} \log{\left (\left\lvert{2 x + 2}\right\rvert \right )}\right]_{3}^{1}$$= \frac{1}{2} \log{\left (\left\lvert{2 x + 2}\right\rvert \right )} + \frac{1}{2} \left\lvert{\log{\left (\left\lvert{2 x + 2}\right\rvert \right )}}\right\rvert$$= \frac{1}{2} \log{\left (p \right )},     \therefore p = 2$ \\
